# 钩子营工具箱

一个基于Claude API的多功能工具箱，包含：

- 🎯 引流标题生成器
- 🎪 谈单回旋镖（销售话术）
- ✍️ 内容加钩子（内容优化）

## 使用方法

1. 输入相关的内容或话题
2. 点击"生成"按钮
3. 查看AI生成的结果
4. 可以复制或下载PDF

## 技术说明

此工具使用Anthropic的Claude 3.5 Sonnet模型，通过提供的API密钥访问AI服务。

## 重要安全说明

⚠️ **注意**：此项目将API密钥直接嵌入前端代码中，这仅供本地开发和测试使用。在生产环境中，你不应该将敏感API密钥暴露在客户端代码中，因为这会导致安全风险。实际部署时应采用后端代理的方式。

## 部署到GitHub Pages

此项目已配置为可通过GitHub Pages访问。

## CORS限制说明

如果直接打开HTML文件，浏览器的CORS策略可能会阻止API请求。通过GitHub Pages访问可以避免此问题。
